
# 基于react-native 的安卓聊天应用

## 主要功能
- [x]   登录注册模块
- [x]   即时通讯模块，私聊及群聊
- [x]   聊天存储及增删
- [x]   用户模块信息模块，搜索用户、即时增添删除好友，
- [x]   文章阅读模块
- [x]   你画我猜模块
